# stock-explorer 개발 세션 — 2026-03-04

## 오늘 한 것

### 1. GitHub Actions 등록
- `.github/workflows/ci.yml`: push/PR 시 pytest 자동 실행 (Python 3.11)
- `.github/workflows/daily-highlights.yml`: 평일 10:00 KST, `--universe --min-signal BUY`
- `.github/workflows/weekly-scan.yml` (Weekend Full Scan): 토요일 10:00 KST, `--universe` 전체 스캔
- 세 워크플로우 모두 `workflow_dispatch`로 수동 실행 가능
- Actions 봇이 결과를 `data/journals/`에 자동 커밋

### 2. Telegram 양방향 봇 추가
fin-advisor의 telegram_bot.py 패턴을 계승하여 stock-explorer 전용 봇 구현.

**push (GitHub Actions → Telegram)**
- 스캔 완료 후 `--notify` 플래그로 결과 자동 전송
- `src/telegram/sender.py`: ExplorationResult → Markdown 포맷 변환 + 전송

**pull (사용자 → 봇)**

| 명령어 | 동작 |
|--------|------|
| `/explore TICKER` / `탐험 TICKER` | 6에이전트 풀 토론 (1~2분) |
| `/top` / `탑픽` | 오늘 BUY+ 저널 요약 |
| `/scan TICKER` / `스캔 TICKER` | 빠른 가격·RSI·MACD 스냅샷 |
| `/help` / `도움말` | 명령어 목록 |

한국어 종목명 지원: `삼성전자`, `하이닉스`, `네이버`, `구글` 등

### 3. 버그 수정
- `/explorer` (r 오타) → `/explore`와 동일하게 처리
- 티커 약칭 자동 교정: `gogl` → `GOOGL`, `goog` → `GOOGL`, `brkb` → `BRK-B` 등

### 4. GitHub 시크릿 등록
```bash
gh secret set TELEGRAM_BOT_TOKEN
gh secret set TELEGRAM_CHAT_ID
```
Actions에서 `--notify` 실행 시 자동으로 시크릿 주입.

## 실행 방법

```bash
cd ~/workspace/stock-explorer

# 봇 실행 (.env 자동 로드)
python scripts/telegram_bot.py

# 수동 스캔 + Telegram 전송
python scripts/explore.py --universe --min-signal BUY --notify

# 테스트
python3 -m pytest tests/ -v
```
