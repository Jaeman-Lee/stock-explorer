### 3. D/E 예외 케이스 보완
**문제**: 애플 D/E 102.6x → 위험 판정 (실제로는 자사주매입 회계 왜곡)

**해결**:
- D/E 대신 **Net Debt / EBITDA** 우선 사용
  - Net Debt < 0 → 순현금 포지션으로 처리 (강점)
  - 애플: Net Debt/EBITDA = 0.2x → 문제없음
- current ratio < 1.0이라도 **FCF마진 > 15%** 면 패널티 경감
- `fundamental_agent.py`, `risk_agent.py` 양쪽 모두 적용
